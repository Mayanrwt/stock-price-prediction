# Stock Price Prediction Model

## 📌 Overview
This project predicts stock prices using machine learning.
## 📊 Output
This model predicts next-day stock prices based on historical data.

## 🛠️ Technologies Used
- Python
- Pandas
- Scikit-learn
- yfinance

## 🚀 How to Run
pip install -r requirements.txt  
python main.py
